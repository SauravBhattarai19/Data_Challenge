THE QUESTION
────────────
Among the 25,142 US census tracts with IGS < 45 in 2017, only 34% crossed that
threshold by 2025. The MS Delta's rate was just 24%. We asked: what, precisely,
separates communities that escaped from those that stayed stuck?


THE FLOW  (follow this order in the app)
─────────────────────────────────────────
1. HOME PAGE
   The problem in numbers. IGS trajectory chart shows the Delta has been ~13
   points below the national mean every year since 2017 — and has never once
   crossed the vulnerability threshold of 45.

2. IGS LANDSCAPE  (Page 1)
   National choropleth. Use the sidebar to filter by state or pillar. This sets
   geographic context: the Delta is not just poor — it is outlier-poor.

3. DELTA DEEP DIVE  (Page 2)
   Select a county (or All Delta) to see every indicator: health outcomes,
   healthcare access, social vulnerability, business landscape, and a tract-level
   Folium map you can color by any variable. The FQHC overlay shows clinic sites.

4. WHAT DRIVES TURNAROUND?  (Page 3)  ← THE KEY PAGE
   This is where the machine learning results live.

   - SHAP Importance chart: 89 features ranked by mean |SHAP|.
     Top finding: DENTAL_CrudePrev (dental visit rate) = 9.2% of total
     predictive power — #1 across all domains, above broadband, income,
     or any housing variable.

   - Beeswarm plot: each dot = one census tract; x-axis = SHAP contribution;
     color = feature value (red = high, blue = low).
     For dental: red dots cluster RIGHT (high dental access → turnaround),
     blue dots cluster LEFT (low access → stuck). Monotone positive effect.

   - Category breakdown: Health Outcomes + IGS sub-indicators = ~50% of all
     SHAP importance. Business Infrastructure is the second-largest domain gap.

5. INVESTMENT PRIORITY MATRIX  (Page 4)  ← THE ACTIONABLE OUTPUT
   Select any of the 9 MS Delta counties from the sidebar (or the full region).
   Each feature gets plotted on two axes:
     X = gap from the turnaround benchmark (county-specific)
     Y = SHAP importance (national, fixed)

   Quadrants:
     ACT NOW     — high importance + large gap       → invest here first
     PROTECT     — high importance + small/no gap    → do not disrupt
     SECOND WAVE — lower importance + large gap      → phase-2
     MONITOR     — lower importance + small gap      → watch and wait

   Thresholds are adjustable. Filter by feature category to focus.


THE HEADLINE FINDING — DENTAL ACCESS
──────────────────────────────────────
Why is dental visit rate the #1 predictor? It is counterintuitive but robust.

  Turnaround tracts average:  56.9%  dental visit rate
  MS Delta average:            6.1%  dental visit rate
  Gap:                        50.8 percentage points
  Delta's national rank:      1st percentile

Spearman correlation (feature value vs. SHAP value) = positive across all 5,000
sampled tracts — the relationship is monotonically positive, not an artifact.

Four reinforcing mechanisms (see Section 3.1 of the report):
  a) Proxy for economic connectivity  — r = 0.469 with Internet Access Score
  b) Healthcare system engagement    — requires transport, insurance, supply
  c) Workforce productivity signal   — dental disease = absenteeism
  d) Income/insurance proxy          — regular dental care needs both

All 9 Delta counties are 100% Primary Care HPSA-designated. The healthcare
infrastructure has essentially collapsed — dental access proxies for that.

Implication: this is an economic infrastructure intervention, not just a
"health" one. Expanding FQHCs, mobile dental units, and Medicaid navigator
programs in the Delta would address the #1 turnaround predictor directly.


MODEL METHODOLOGY (brief)
──────────────────────────
Study population : 25,142 tracts with IGS < 45 in 2017
Outcome          : Turnaround = IGS ≥ 45 by 2025 (n = 8,547 yes, 16,595 no)
Features         : 89 granular features across 10 public datasets
                   (population-normalised rates, NOT opaque composites)

Models trained with 5-fold stratified CV:
  Logistic Regression   AUC 0.718 ± 0.006  (linear baseline)
  Random Forest         AUC 0.763 ± 0.005  ← SHAP source
  Gradient Boosting     AUC 0.771 ± 0.004

TreeSHAP run on 5,000 randomly sampled tracts (random_state = 42).
Direction labels corrected using Spearman corr(feature_value, SHAP_value)
rather than sign of mean SHAP — avoids sign artifacts on near-zero means.

CLONE AND RUN LOCALLY
──────────────────────
All processed data files are committed — no raw data required to run the app.

  Step 1 — Clone
  ──────────────
  git clone https://github.com/SauravBhattarai19/Data_Challenge.git
  cd Data_Challenge

  Step 2 — Install dependencies  (Python 3.10+)
  ─────────────────────────────
  pip install -r requirements.txt

  Step 3 — Launch
  ───────────────
  cd project/04_app
  streamlit run app.py

  The app opens at http://localhost:8501
  All four pages are immediately usable — no pipeline run needed.

  Optional: reproduce the report figures
  ──────────────────────────────────────
  cd Data_Challenge          # back to repo root
  python project/report/generate_figures.py
  cd project/report && pdflatex report.tex

  Optional: retrain the ML model from processed data
  ────────────────────────────────────────────────────
  python project/03_analysis/expanded_priority_matrix.py
  # Writes expanded_model.parquet, expanded_shap_*.parquet
  # (~5 min; requires scikit-learn + shap already installed above)

  Requirements snapshot (key packages)
  ─────────────────────────────────────
  streamlit >= 1.35       plotly >= 5.18      scikit-learn >= 1.4
  pandas >= 2.0           folium >= 0.15      shap >= 0.45
  numpy >= 1.24           streamlit-folium    joblib >= 1.3
